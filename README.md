# 📝 Basic Todo App
A beginner-friendly Todo application built with **Flutter**.  
This app allows users to **add, update, and delete tasks** easily.
## 🚀 Features
- Add new tasks  
- Mark tasks as completed  
- Delete tasks  
- Simple and clean UI
